### 4. ✅ 添加到知识图谱 (Add to Knowledge Graph)
**测试内容**: 通过 POST `/api/memories/{id}/add-to-graph` 将记忆加入知识图谱

**测试结果**:
- ✅ HTTP 状态码: 202 Accepted
- ✅ 返回队列确认消息
- ✅ 图谱状态设置为 `pending`
- ✅ 记忆 ID 加入异步处理队列

**后台处理**:
- GraphitiIngestWorker 从队列中取出任务
- 调用 Graphiti SDK 处理记忆内容
- 使用 StepFun API 进行 LLM 处理
- 使用本地 embedding 模型生成向量
- 将结果存储到 Neo4j 知识图谱

---

### 5. ✅ 查询图谱状态 (Check Graph Status)
**测试内容**: 通过 GET `/api/memories/{id}/graph-status` 查询图谱处理状态

**测试结果**:
- ✅ HTTP 状态码: 200 OK
- ✅ 返回图谱状态信息
- ⏳ 状态: `pending` (20秒后仍在处理中)

**状态说明**:
- `not_added`: 未添加到图谱
- `pending`: 正在处理中
- `added`: 成功添加,包含 episode_uuid
- `failed`: 处理失败,包含错误信息

**注意**: 图谱处理是异步的,可能需要 10-60 秒完成,取决于:
- 内容复杂度
- StepFun API 响应时间
- Neo4j 写入速度
