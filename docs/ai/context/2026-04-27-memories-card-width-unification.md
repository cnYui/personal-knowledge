# 2026-04-27 memories 卡片宽度统一

## 背景

- `memories` 页面当前列表项只设置了 `maxWidth`，未显式设置 `width`
- 在 `display: flex` 的包裹容器里，部分内容较短的卡片会按内容收缩，导致列表长度不一致

## 决策

- 保持现有视觉结构不变
- 仅为 `MemoryBubbleItem` 增加 `width: 100%`
- 继续保留 `maxWidth: { xs: '100%', md: '82%' }`，这样桌面端仍维持原本的阅读宽度上限

## 结果

- 所有记忆卡片按同一宽度规则展示
- 短内容卡片不再因为内容长度不同而出现明显长短不一
